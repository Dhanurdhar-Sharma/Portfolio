Smart Home Maker

Smart Home Maker is a DIY automation project that transforms a regular home into a smart home using microcontrollers (like Arduino or ESP32), and basic electronics. It enables control of lights, fans, and other appliances through smartphone apps. The project focuses on energy efficiency, user convenience, and affordable home automation solutions for beginners and enthusiasts.


Key Features:

  • Device Control: Automate and control appliances like lights, fans, and alarms using mobile app
  • Sensor Integration: Use temperature sensors to create intelligent automation rules.
  • User Management: Allow multiple users with access control and customization of preferences.
  • Auto Mode: The system senses your room temperature and automatically turns the AC on or off accordingly, helping save energy.
  • Responsive Dashboard: A user-friendly and responsive interface for real-time monitoring and control from any device.


Technologies Used:

  • App Development: Flutter.
  • Hardware Used: ESP32, 4 channel Relay Module
  • Tools Used: Android Studio, FireBase, Arduino IDE
